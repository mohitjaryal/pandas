# 🐼 Pandas Learning Repository

> A hands-on collection of Jupyter Notebooks for learning **Pandas** — the most powerful Python library for data analysis and manipulation.

---

## 📌 About This Repository

This repository is a beginner-to-intermediate guide to mastering **Pandas** in Python. It contains well-structured Jupyter Notebooks covering everything from the basics of Pandas to working with DataFrames — the core building block of data analysis in Python.

Whether you're just getting started with data science or brushing up on your Pandas skills, this repo has you covered.

---

## 📁 Repository Structure

```
pandas/
│
├── 📓 basic_pandas.ipynb       # Core Pandas concepts and operations
├── 📓 data_frames.ipynb        # Deep dive into DataFrames
├── 📂 data/                    # Dataset files used in the notebooks
└── 📄 README.md                # You are here!
```

---

## 📓 Notebooks Overview

### 1. `basic_pandas.ipynb` — Pandas Basics
An introduction to the Pandas library covering essential concepts:

- ✅ Importing Pandas
- ✅ Pandas **Series** — creating and manipulating 1D data
- ✅ Reading data from CSV, Excel, and other formats
- ✅ Inspecting data: `.head()`, `.tail()`, `.info()`, `.describe()`
- ✅ Indexing and selecting data
- ✅ Filtering rows with conditions
- ✅ Handling missing values (`NaN`)
- ✅ Basic data operations — sorting, renaming, dropping columns

---

### 2. `data_frames.ipynb` — Working with DataFrames
An in-depth exploration of Pandas **DataFrames**:

- ✅ Creating DataFrames from dictionaries, lists, and CSV files
- ✅ Accessing rows and columns using `.loc[]` and `.iloc[]`
- ✅ Adding and removing columns
- ✅ Filtering and querying DataFrames
- ✅ Grouping data with `.groupby()`
- ✅ Merging, joining, and concatenating DataFrames
- ✅ Applying functions with `.apply()`
- ✅ Pivot tables and aggregation
- ✅ Exporting DataFrames to CSV/Excel

---

### 3. `data/` — Datasets
This folder contains all the raw dataset files (`.csv`, `.xlsx`, etc.) used across the notebooks for real-world practice examples.

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed. Then install the required libraries:

```bash
pip install pandas numpy jupyter
```

### Run the Notebooks

1. **Clone this repository:**

```bash
git clone https://github.com/mohitjaryal/pandas.git
cd pandas
```

2. **Launch Jupyter Notebook:**

```bash
jupyter notebook
```

3. **Open any notebook** from the browser interface and start learning! 🎉
